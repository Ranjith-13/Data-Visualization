A small Data Visualization project of a Dashboard using Tableau 

Netflix Movies and TV Shows Dashboard

This Dashboard contains general info of Netflix from its beginning to recent years.
Infos are in the form of Data Visualization (i.e., like Top 10 Genres, Distribution)

It is also a little interactive.
After mentioning the Type(Movies/TV Shows) and Title of the Show it displays info like the Genre, Duration, Description, etc., of the show.

The Source Data Set is taken from Kaggle: https://www.kaggle.com/code/chirag9073/netflix-data-analysis/data?select=netflix_titles.csv
